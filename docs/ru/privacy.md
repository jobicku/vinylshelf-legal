# ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ

**VinylShelf**

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты информации пользователей (далее — «Пользователь») мобильного приложения VinylShelf (далее — «Приложение») на территории Российской Федерации.

## 1. ОБЩИЕ ПОЛОЖЕНИЯ

1.1. Использование Приложения означает согласие Пользователя с условиями настоящей Политики.

1.2. Настоящая Политика разработана в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

1.3. Владелец вправе вносить изменения в Политику в одностороннем порядке. Действующая редакция размещается в Приложении.

## 2. ПРИНЦИПЫ ОБРАБОТКИ ИНФОРМАЦИИ

2.1. Приложение не собирает и не обрабатывает персональные данные Пользователя в понимании Федерального закона № 152-ФЗ.

2.2. Владелец не имеет возможности идентифицировать Пользователя на основании собираемой информации.

2.3. Вся информация хранится исключительно на устройстве Пользователя в локальном хранилище.

2.4. Приложение не использует файлы cookie и аналогичные технологии отслеживания.

## 3. ИСПОЛЬЗОВАНИЕ СТОРОННИХ СЕРВИСОВ

### 3.1. Discogs API

3.1.1. Приложение использует Discogs API для получения справочной информации о музыкальных релизах и не является аффилированным с Discogs.

3.1.2. При обращении к Discogs API не передаются данные, позволяющие идентифицировать Пользователя.

3.1.3. Обработка данных сервисом Discogs регулируется политикой конфиденциальности Discogs, размещенной на сайте https://support.discogs.com/hc/en-us/articles/360009334513-Privacy-Policy.

### 3.2. Yandex AppMetrica

3.2.1. Приложение использует сервис Yandex AppMetrica для сбора обезличенной аналитической информации.

3.2.2. Yandex AppMetrica собирает следующие данные:
- тип и модель устройства;
- версия операционной системы;
- данные об использовании функций Приложения;
- информация о сбоях и ошибках;
- IP-адрес устройства (обезличенный и неиспользуемый для идентификации Пользователя);
- уникальный обезличенный идентификатор установки (AppMetrica Device ID), используемый для аналитики и диагностики;
- обезличенные данные о взаимодействии с релизами.

3.2.3. Собираемые данные не позволяют идентифицировать Пользователя.

3.2.4. Обработка данных сервисом Yandex AppMetrica регулируется политикой конфиденциальности ООО «ЯНДЕКС», размещенной на сайте https://yandex.ru/legal/confidential/ru/.

### 3.3. Tracer

3.3.1. Приложение использует Tracer для технического мониторинга и отладки.

3.3.2. Tracer собирает обезличенные технические данные о работе Приложения без возможности идентификации Пользователя.

3.3.3. Обработка данных сервисом Tracer регулируется политикой конфиденциальности ООО «ВК», размещенной на сайте https://apptracer.ru/privacy/.
