# Политика конфиденциальности (RU)
**Версия:** 1.0.0  
**Дата вступления в силу:** 30.10.2025

_Краткое резюме:_ Приложение хранит все пользовательские данные локально; для аналитики используются AppMetrica и Firebase Crashlytics; данные об альбомах получаем из Discogs API.

## 1. Кто мы
VinylShelf — персональное мобильное приложение для управления коллекцией винила.

## 2. Обрабатываемые данные
- Локальные данные: коллекция, заметки — **только на устройстве**.
- Технические данные аналитики: см. раздел «Провайдеры».
